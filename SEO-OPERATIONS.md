# SEO Operations - K1 Compare Lab

Primary keyword: k1 game alternative

## Launch tasks

- Submit /sitemap.xml to Google Search Console.
- Add GA4 and Search Console verification after final domain is attached.
- Request indexing for homepage, four hub pages, /guides/, and the first six guide pages.
- Add original screenshots before publication.

## Weekly sprint

- Publish or refresh 2-5 pieces.
- Check GSC queries by page.
- Improve titles when CTR is low after 200 impressions.
- Add FAQ answers from new query data.
- Check 404, canonical, mobile usability, and Core Web Vitals.

## Ranking phases

- Phase 1: indexing stable.
- Phase 2: long tails top 50.
- Phase 3: strongest guides top 20.
- Phase 4: hub and core terms top 10/top 3.
